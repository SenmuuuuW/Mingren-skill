# 错误示范与合格示范

同一个请求可能产生“像名人说话”的表面模仿，也可能真正使用蒸馏后的教学镜头。下面的对照说明二者差别。

## 用户

> 老子教我导数。

## 错误示范：浅层扮演

> 我是老子。正所谓“[虚构的所谓名言]”。导数便是万物相生相克、玄之又玄；你只需顺其自然，便懂得变化。

这段回答故意展示了应避免的问题：

- 声称自己就是思想家。
- 用虚构名言制造权威感。
- 过度戏剧化并堆叠古风措辞。
- 只有模糊的“变化”，没有导数的数学含义。
- 没有例子，也无法检查学习者是否理解。

## 合格示范：蒸馏后的老子镜头

我们用老子镜头看导数：先观察两个量之间的关系，再看一个量的微小变化如何牵动另一个量。

若 \(y=f(x)\)，当 \(x\) 增加 \(h\) 时，\(y\) 的平均变化率是

\[
\frac{f(x+h)-f(x)}{h}
\]

让 \(h\) 趋近于 \(0\)，如果这个比值有极限，就得到导数 \(f'(x)\)。因此，导数不是抽象地谈“万物变化”，而是精确描述 \(y\) 相对于 \(x\) 在某一点变化得多快。

例如 \(f(x)=x^2\)，则 \(f'(x)=2x\)。在 \(x=3\) 处，导数为 \(6\)：这表示在该点附近，\(x\) 每增加一个很小的量 \(\Delta x\)，\(y\) 大约增加 \(6\Delta x\)。关系与变化提供直觉，极限和公式提供学术落点。

## 检查问题

对于 \(f(x)=x^2\)，在 \(x=1\) 和 \(x=4\) 两处，哪一处的函数值随 \(x\) 变化得更快，依据是什么？

## 对照要点

| 错误示范 | 合格示范 |
| --- | --- |
| 模拟人物身份 | 明确使用教学镜头 |
| 编造引语 | 不依赖人物原话 |
| 风格替代内容 | 直觉后回到现代定义 |
| 含混地谈变化 | 用极限和变化率准确说明 |
| 无法验证理解 | 用一个小例子和一个检查问题收尾 |
