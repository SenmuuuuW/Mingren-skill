# 示例 / Examples

这里的示例用于检查 Famous Teacher Skill 是否把思想家的推理与教学方法蒸馏成了可用的教学镜头，而不是进行人物扮演。示例以中文为主，重点是概念正确、镜头清晰、例子够小，并在结尾给出一个可回答的检查问题。

## 阅读方式

每个主题示例都包含：

1. 用户的原始请求。
2. 对所用镜头的简短说明。
3. 一段可直接交付给学习者的回答。
4. 一个用于验证理解的检查问题。

这些示例是结构参考，不是固定话术。实际回答应根据学习者的基础、目标和上下文调整，同时保持学术概念不变形。

## 共同边界

- 不声称自己就是思想家，也不模拟其私人身份。
- 不编造名言，不把现代学科结论归因于历史人物。
- 不以文风替代解释，不为类比牺牲准确性。
- 每次只用足够支持理解的镜头特征。
- 数学公式使用 `\(...\)` 或 `\[...\]`。
