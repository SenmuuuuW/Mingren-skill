# 冯·诺伊曼镜头：神经网络

## 用户

> 冯诺伊曼教我神经网络。

## 镜头说明

下面借用输入、状态、规则、输出和模块分解来建立计算模型。这是蒸馏后的系统镜头，不表示冯·诺伊曼本人对现代神经网络作过这些具体表述。

## 示例回答

我们用冯·诺伊曼镜头，把神经网络看成一个分层的变换系统。

**输入**：原始数据 \(x\)，例如一张图片的像素或一句话的词向量。

**参数与中间量**：权重 \(W\) 和偏置 \(b\) 是可训练参数；每层产生的 \(z\) 和激活值 \(a\) 是一次前向计算的中间量。普通前馈网络在单次推理中通常没有随时间演化的内部状态；训练系统的持久状态还可能包括优化器状态。

**规则**：一层先做线性变换，再通过非线性激活函数：

\[
z^{(l)}=W^{(l)}a^{(l-1)}+b^{(l)},\qquad
a^{(l)}=\sigma\!\left(z^{(l)}\right)
\]

多层网络就是把这个模块连续组合。非线性函数 \(\sigma\) 很重要；如果每层都只有线性变换，无论叠多少层，整体仍只是一个线性变换。

**输出**：最后一层给出预测，例如类别概率或一个数值。

**训练更新**：训练时，损失函数先比较预测与目标。反向传播计算每个参数对损失的影响，优化器再更新 \(W\) 和 \(b\)。所以一次训练循环可以写成：输入数据 → 按规则前向计算 → 产生输出与损失 → 反向计算 → 更新参数与优化器状态。

看一个最小模块。令输入 \(x=(1,2)\)，权重 \(W=(0.5,-0.25)\)，偏置 \(b=0.1\)，激活函数为 ReLU，则

\[
z=0.5\times1-0.25\times2+0.1=0.1,
\qquad a=\max(0,z)=0.1
\]

这一神经元把两个输入按固定规则变成一个输出；训练改变的是规则中的参数，而不是输入本身。
