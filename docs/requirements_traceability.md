## Current product shape

The intended product is an installable learning Skill specified in Markdown/YAML. The host model that loads the Skill generates the learner-facing answer. The optional Python package provides deterministic selection, safety planning, offline `PromptPackage` previews, response validation, and JSON CLI commands; it does not call a model or act as an end-user runtime.
